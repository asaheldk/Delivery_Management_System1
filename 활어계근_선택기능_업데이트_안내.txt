장백수산 배송원 페이지 V9.2 · 활어 계근 선택 기능 안내

[변경 목적]
기존에는 활어 계근을 모두 입력하고 확정해야 거래명세서 인쇄·사진 저장과
하차 완료가 가능했습니다. 배송원 업무 지연과 불편을 줄이기 위해 이 강제 조건을 제거했습니다.

[변경된 내용]
1. 활어 계근을 하지 않아도 거래명세서가 바로 표시됩니다.
2. 계근 여부와 관계없이 인쇄 버튼을 사용할 수 있습니다.
3. 계근 여부와 관계없이 PNG 저장과 JPG 저장을 사용할 수 있습니다.
4. 계근 미확정 때문에 출차 최종 점검이 차단되지 않습니다.
5. 계근 미확정 때문에 하차 완료가 차단되지 않습니다.
6. 활어 계근 기능 자체는 삭제하지 않고 선택 기능으로 유지합니다.
7. 계근을 입력한 경우 기존처럼 마릿수와 kg이 거래명세서에 자동 반영됩니다.

[그대로 유지되는 확인 기능]
- 수조 운반 품목의 실제 수조 상차 확인
- 박스 및 동봉품 확인
- 거래처별 하차 품목 체크
- 배송 완료 취소와 내용 수정
- 전미수 수정

[배송원이 사용하는 방법]
1. 배송지를 누릅니다.
2. 하차할 어종과 박스를 확인합니다.
3. 거래명세서를 바로 인쇄하거나 PNG/JPG로 저장합니다.
4. 모든 하차 물품을 확인한 후 하차 완료를 누릅니다.
5. 활어 계근이 필요한 날에만 ‘호차 활어 계근 · 선택’을 사용합니다.

[업데이트 방법]
- GitHub의 driver.html만 새 파일로 교체합니다.
- admin.html과 Supabase SQL은 다시 교체하거나 실행할 필요가 없습니다.
- GitHub 저장 후 Vercel 최신 배포가 Ready가 되면 배송원 페이지를 새로고침합니다.
