장백수산 관리자 V2.4.2 계정 관리 사용 안내
========================================

이번에 추가된 기능
----------------
1. 최고 관리자가 본인 계정을 화면에서 직접 수정
2. 최고 관리자가 다른 관리자의 이름·아이디·역할·상태·허용 메뉴 수정
3. 최고 관리자만 다른 관리자 계정 삭제
4. 상단 '업데이트 새로고침' 버튼으로 최신 배포 파일 다시 불러오기
5. 관리자 생성·수정·삭제 내용 감사 기록 저장


1. 업데이트 적용 방법
-------------------
1) V2.4.2 ZIP 파일의 압축을 풉니다.
2) 압축을 푼 폴더 안의 파일과 폴더를 GitHub 저장소에 덮어씁니다.
3) api 폴더와 api/admin 폴더의 위치를 바꾸면 안 됩니다.
4) Vercel의 최신 배포가 초록색 Ready인지 확인합니다.
5) 관리자 페이지에 다시 접속합니다.
6) 왼쪽 위에 OPERATIONS CENTER V2.4.2가 보이면 적용 완료입니다.

이번 기능은 Supabase 표를 새로 만들지 않으므로,
V2.4.1 역할 오류 수정 SQL을 이미 실행했다면 추가 SQL 실행은 필요 없습니다.


2. 업데이트 새로고침 버튼 사용
----------------------------
로그인 후 화면 오른쪽 위에 다음 버튼이 보입니다.

↻ 업데이트 새로고침

이 버튼을 누르면:
- 이전 화면 캐시를 정리합니다.
- Vercel에 새로 배포된 admin.html을 다시 불러옵니다.
- 주소 뒤에 새 갱신 번호를 붙여 이전 파일 사용을 방지합니다.

Vercel 배포가 Ready가 되기 전에 누르면 이전 버전이 나올 수 있습니다.
반드시 Vercel에서 Ready를 확인한 뒤 누르세요.

아이폰에서는 버튼 공간을 줄이기 위해 ↻ 모양만 표시될 수 있습니다.
기능은 컴퓨터와 같습니다.


3. 본인 최고 관리자 계정 직접 수정
--------------------------------
1) 상단 메뉴에서 '권한·감사'를 누릅니다.
2) 관리자 계정 표에서 본인 행을 찾습니다.
3) 오른쪽의 '내 계정 설정'을 누릅니다.
4) 다음 내용을 수정할 수 있습니다.

- 관리자 아이디
- 담당자 이름
- 상단에 표시할 허용 메뉴

5) '내 계정 저장'을 누릅니다.

안전 보호:
- 본인의 최고 관리자 역할은 이 화면에서 변경할 수 없습니다.
- 본인의 사용 상태를 정지할 수 없습니다.
- 다시 권한 화면에 들어올 수 있도록 '권한·감사' 메뉴는 해제할 수 없습니다.
- 본인 계정은 삭제할 수 없습니다.

중요:
관리자 아이디를 바꿔도 로그인 이메일은 바뀌지 않습니다.
관리자 로그인은 기존 이메일과 비밀번호를 계속 사용합니다.


4. 다른 관리자 메뉴·역할 커스텀
------------------------------
1) '권한·감사'를 누릅니다.
2) 수정할 관리자의 오른쪽 '권한·메뉴 수정'을 누릅니다.
3) 다음 항목을 수정합니다.

- 관리자 아이디
- 담당자 이름
- 역할
- 계정 상태
- 허용 메뉴 체크박스

4) 역할을 바꾸면 추천 메뉴가 즉시 자동 체크됩니다.
5) 필요 없는 메뉴는 체크를 해제합니다.
6) 추가할 메뉴는 직접 체크합니다.
7) '저장'을 누릅니다.

이 기능은 최고 관리자 또는 기존 전체 관리자만 사용할 수 있습니다.
배차·회계·현장·조회 관리자는 다른 관리자의 권한을 변경할 수 없습니다.

권한이 변경된 관리자는 로그아웃 후 다시 로그인하면
변경된 상단 메뉴를 정확하게 확인할 수 있습니다.


5. 다른 관리자 삭제
------------------
1) '권한·감사'를 누릅니다.
2) 삭제할 관리자 행 오른쪽의 빨간색 '삭제'를 누릅니다.
3) 삭제 확인 창에서 대상 이름을 다시 확인합니다.
4) 확실할 때만 확인을 누릅니다.

삭제하면:
- 해당 관리자의 로그인 계정이 삭제됩니다.
- 관리자 목록에서 사라집니다.
- 해당 이메일과 비밀번호로 로그인할 수 없습니다.
- 삭제 작업은 감사 기록에 남습니다.

주의:
- 삭제는 되돌릴 수 없습니다.
- 퇴사했지만 기록 보존이 필요한 경우에는 삭제보다
  '권한·메뉴 수정 → 계정 상태 → 사용 정지'를 권장합니다.
- 현재 로그인한 본인 계정은 삭제할 수 없습니다.


6. 버튼이 보이지 않을 때
-----------------------
다음 순서로 확인하세요.

1) 왼쪽 위 버전이 V2.4.2인지 확인
2) Vercel 최신 배포가 Ready인지 확인
3) 오른쪽 위 ↻ 업데이트 새로고침 누르기
4) 로그아웃 후 다시 로그인
5) 로그인 계정 역할이 최고 관리자인지 확인

V2.4.1 또는 이전 버전이 보인다면 GitHub에 새 admin.html이
정상적으로 올라가지 않았거나 Vercel 배포가 아직 끝나지 않은 것입니다.
